HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • <h1-6>: levels of heading (in descending order)
  • <p>: paragraph/block of text
  • <li>: list items
  • <ul>: unordered list
  • <ol>: ordered list
  • <img>: contains information about images displayed on webpage
  • <br>: creates a line break (line of empty space) between blocks of content
  • <a>: anchor; creates links to the same webpage/other webpages
